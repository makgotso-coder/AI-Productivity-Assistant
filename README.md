# AI-Productive-Assistant-
AI-powered assistant that automates workplace tasks.

###Features
1.**Email Generator**;Create professional emails from bullet points
2.**Meeting Summaries**;Turns transcripts into key points+action items
3.**Task planner**;Breaks goals into weekly tasks

##How to Use
1.Copy the prompts from"prompts"folder
2.Paste into ChatGPT/Gemini/Notion AI 
3.Replace [brackets] with your info

##Example
1.Input:"Summaries this meeting meeting:[paste transript]"
Output:[show 3 bullet points+2 action items]

###Ethical Use
This tool assists,not replaces human judgement.Always review outputs.
